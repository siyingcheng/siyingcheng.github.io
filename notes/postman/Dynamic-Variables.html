Dynamic Variables PostmanNotes ​

Postman uses the faker library to generate sample data, including random names, addresses, email addresses, and much more. You can use these pre-defined variables multiple times to return different values per request.

And here is the official documentation. Dynamic variables list

💡 Use Dynamic Variables in scripts or pre-request

To use dynamic variables in pre-request or test scripts, you need to use  pm.variables.replaceIn(), for example  pm.variables.replaceIn('').
